# 🧮 simple-calculator
Design & Interaction Breakdown

This calculator implements cutting-edge design trends with separate files:

1. Light/Dark Theme System

    Toggle between themes with the floating button

    All colors, gradients, and shadows adapt seamlessly

    Theme preference saved in localStorage

2. Creative JavaScript Interactions

    Custom Cursor: Dot and ring follow mouse with smooth lerp animation

    Magnetic Buttons: Buttons subtly pull toward cursor on hover with rotation

    Ripple Effect: Click creates expanding ripple animation

    Keyboard Support: Full keyboard navigation with visual feedback

    Smooth Animations: Display scales on calculation result

3. Performance & Efficiency

    CSS will-change optimized for smooth animations

    Throttled resize handler

    No external libraries (only Font Awesome for icons)

    Reduced motion support for accessibility
